Desafio Terra!

Proposta do desafio

Criar uma api de anticorrupção para expor serviços do github:

  • API de REST do GitHub - Documentação do GitHub

A partir da documentação, consumir e disponibilizar na camada anti-corrupção os seguintes serviços::

  • Criação de repositório;
  • Listar branchs de um repositório;
  • Listar webhooks de um repositório;
  • Adicionar webhook a um repositório;
  • Atualizar uma webhook de um repositório;

Requisitos

Obrigatórios

  • API Rest (funcionando como camada anti-corrupção consumindo API externa);
  • Uso do swagger para documentação;

Desejáveis

  • .Net 6
  • Testes unitários
  • Dependency Injection
  • SOLID

Orientações

Deverá ser enviado o link do repositório no Github com o código para o email b2c.desenvolvimento@terrainvestimentos.com.br assim como as devidas orientações para rodar o projeto.
